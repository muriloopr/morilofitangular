Sobre o Projeto

Este projeto foi desenvolvido como parte de um trabalho da Etec, com o objetivo de aplicar conhecimentos em Angular, JSON Server e criação de aplicações web modernas.

Tecnologias Usadas

  • Angular 17
  • JSON Server
  • TypeScript
  • HTML + CSS
  • Angular Material

Funcionalidades

  • Login de usuários
  • Envio e visualização de mensagens
  • Modo escuro personalizável
  • Configurações de acessibilidade (animações)
